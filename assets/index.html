Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • Code contained within the head is visible to computers, not end users.
  • The body element represents the visible content shown to the user.
  • Code within the body defines the content and styling visible to the end user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • In-line notes place css code inside html sections.
  • An External CSS Style Sheet creates a to an external page in the head.
  • "rel" indicates the relationship between the linked information and the html page.
  • "href" indicates the location of the information.
  • In the Command Line, type cd assets, then touch "style.css" to create a linkable css file.
  • Selectors define which html elements css code applies to.
  • Use a comma to apply two selectors to a css rule {}.
  • Class elements allow CSS to apply only to specified elements within the html code.

Git

  • git status = checks what branch we are currently on
  • git checkout -b branch-name = creates a new branch and switches to it
  • git add -A = add new code to branch
  • git commit -m "fixed bug in..." = commit branch changes to main
  • git pull origin main = pull down latest main branch
  • git push origin feature/starter-code = send to online github
